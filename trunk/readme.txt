HTML Component Framework


DEPLOYMENT NOTES
Use the ANT build file to create a distribution in a folder "output".
The deployment requires yuicompressor-2.4.2.jar in the folder "library/java" - http://developer.yahoo.com/yui/compressor/


USE
See http://code.google.com/p/htmlcomponentframework/
The framework depends on the 3rd party JavaScript libraries:
jquery-1.3.2.min.js - http://jquery.com/
json2.js - http://www.json.org/json2.js
jsonschema-b4.js - http://code.google.com/p/jsonschema/
